# Food Express App — UI/UX Case Study 📱🍲

An end-to-end UX Research, User Flow logic architecture, and Low-Fidelity Wireframing project. This case study focuses on creating a seamless, accessible solution for working professionals and students to source affordable, healthy, and hygienic home-cooked meal options.

---

## 📁 Project Deliverables
This repository contains the following structural design artifacts:
1. **`UX_Research_Documentation.pdf`** — Complete user empathy analysis, targeting our primary user persona (Ananya R) along with an end-to-end User Journey Map.
2. **`User_Flow_Diagram.png`** — System logic mapping and structural architecture created using Draw.io to visualize structural route navigation logic.
3. **`Figma_Wireframe_Screens/`** — High-quality exports of the 5-screen interactive low-fidelity grayscale wireframes built using structural grid layouts.

---

## 📱 Wireframe User Flow & Screens Included
The interactive low-fi wireframe covers 5 core steps mapping the entire transaction journey:
* **Frame 1: Splash Screen** — Core application onboarding block featuring the branding focus.
* **Frame 2: Login Screen** — Input entry block displaying explicit username, password placeholders, and structural call-to-action button setup.
* **Frame 3: Dashboard** — Categorized search console with responsive filters and recommended restaurant preview grid lists.
* **Frame 4: Express Menu List & Cart** — Custom menu list detail section complete with item quantifiers, pricing tiers, and direct cart actions.
* **Frame 5: Profile & Order Tracking** — Real-time active delivery layout configuration mapping dispatch metrics alongside direct rider connectivity modules.
